# Expo Project

Look at the [Expo documentation](https://docs.expo.dev/) to learn more.

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install
```

## Run App

Start the development server on `http://localhost:3000`:

```bash
# run options
npm run start

# run on android
npm run android

# run on ios
npm run ios

# run on web
npm run web

```
